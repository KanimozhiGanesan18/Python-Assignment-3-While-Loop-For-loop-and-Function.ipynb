📖 Overview

This repository contains Python programming exercises focused on While Loops, For Loops, and Functions. The notebook includes beginner-friendly tasks that demonstrate looping, control statements, user input handling, and function creation through practical examples.


🚀 Tasks Included

🎮 Task 1: Number Guessing Game

Uses a while loop and control statements (break and continue)
Generates a random number between 1 and 10
Validates user input
Provides hints for high/low guesses
Limits the number of attempts

🔢 Task 2: Multiplication Table

Uses a for loop to generate the multiplication table (1–10)
Accepts user input for any number
Displays the multiplication table in a formatted output

⚖️ Task 3: BMI Calculator

Defines a reusable Python function calculate_bmi()
Accepts weight and height as user input
Calculates Body Mass Index (BMI)
Displays the result rounded to two decimal places

🎯 Learning Objectives

By completing this notebook, you will learn how to:

Use while and for loops effectively.
Apply control statements such as break and continue.
Create and call Python functions.
Handle user input and perform calculations.
Build simple interactive Python programs.
